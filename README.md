# BooksApp - Frontend 

El cliente de **BooksApp**, una aplicación para gestionar tu biblioteca personal y recibir recomendaciones y organizar tus lecturas. Usa **React**, **Chakra UI** y **Vite**.

##  Características actuales
* **Autenticación JWT:** Integración con el backend de Spring Boot.
* **Context API:** Gestión global del estado del usuario (token, roles, perfil).
* **Navegación:** Rutas protegidas mediante React Router.

##  Tecnologías utilizadas
* **Chakra UI** (Sistema de diseño y componentes)
* **React Router Dom** (Navegación)
* **Context API** (Estado global)
* **Fetch API** (Consumo de servicios)
